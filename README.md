## Deployment on Vercel (recommended)

This project is optimized for **zero-config deployment** on Vercel.

### Quick deploy

1. Push the folder to GitHub / GitLab
2. Import the repository in [Vercel](https://vercel.com)
3. Vercel automatically detects the static site and uses `vercel.json`
4. Your site is live in under 30 seconds

### What the config does

- `vercel.json` ‚Üí Enables clean URLs, adds enterprise-grade security headers (important for a trust/verification product)
- `.vercelignore` ‚Üí Excludes unnecessary files from the production build

### Custom domain

After deployment, go to your project ‚Üí Settings ‚Üí Domains and add your custom domain (e.g. `veritaslayer.com`).
